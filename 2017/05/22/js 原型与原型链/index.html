原型与原型链

发表于 2017-05-22

一、原型

1.1、简介

每一个对象都有一个原型，通过构造函数的 prototype 可以访问到原型，一个对象如果要查找一个属性，首先会在自身的作用域上查找，如果找不到，就到原型上去查找，还找不到，就会报错。所以一般对象有公共的方法就可以放在原型上。

二、原型链

在javascript中，每个对象都有一个指向它的原型(prototype)对象的内部链接。每个原型对象又有自己的原型，直到某个对象的原型为null为止，组成这条链的最后一环。
