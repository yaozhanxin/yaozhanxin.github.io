一、HBuilder 打包 Vue-cli 构建的 APP

1.1

用 Vue-cli 构建好 vue 项目，在打包之前需要修改一个文件

config 下的 index.js 文件，将 assetsPublicPath: '/' 修改为 assetsPublicPath: './'

1.2

执行打包命令

打包成功后会生成一个 dist 文件夹

在 HBuilder 中打开 dist 文件夹，在该项目上右击，选择转换成移动 App

转换成功后会生成 manifest.json 配置文件，在里面配置 APP 的一些信息

配置完毕后, 在 HBuilder 中点击 “发行” 按钮 – 发行为原生安装包

最后会将 dist 提交到云端，在云端打包完毕后下载到本地即可
